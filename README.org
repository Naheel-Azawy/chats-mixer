* CHATS-MIXER
  Mixes chats from different services into a single JSON and a compressed and encrypted HTML.

** Example
  [[./screenshots/ab.png]]
  HTML and JSON versions can be found under `sample-out/`

** Supported services
   - Whatsapp
   - Facebook
   - Instagram
   - Telegram
   - SMS
